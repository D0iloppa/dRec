## 🏗️ 인프라 구조

```
doil.me (Nginx Gateway)
├── /          → doil-react (React, Vite)
├── /wiki/     → doil-wiki (Docusaurus)
├── /sb/       → doil-sb (Next.js)
├── /cdn/      → Static CDN (imgproxy)
├── ohno.doil.me    → Oh!NO 서비스
├── saigon.doil.me  → SaigonRider
├── doybrary.doil.me → Kavita (도서관)
└── jenkins.doil.me  → CI/CD
```

**운영 환경:** Ubuntu Server + Docker Compose + Certbot (Let's Encrypt)
