## 발급 후 — nginx 적용 (필수)
certbot 은 `/etc/letsencrypt/live/doil.me/` 에 저장하지만, nginx 는 `./nginx/live/doil.me/` 를 마운트해서 읽는다.
발급/갱신 후 아래로 복사해야 새 인증서가 반영된다.
```
sudo cp -L /etc/letsencrypt/live/doil.me/fullchain.pem /etc/letsencrypt/live/doil.me/privkey.pem /mnt/c/DEV/docker/nginx/live/doil.me/
docker compose exec doil-gw nginx -s reload
```

## webroot 챌린지 주의
HTTP-01 챌린지는 `http://<도메인>/.well-known/acme-challenge/` 로 들어온다.
백엔드로 프록시되는 서브도메인(plane, docs 등)은 `:80` 블록에서 이 경로를
`return 301` 보다 먼저 `root /app/html;` 으로 예외 처리해야 검증이 통과한다.
(예외가 없으면 https 로 리다이렉트→백엔드 프록시되어 챌린지 파일을 못 찾고 무한 대기)
